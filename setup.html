Vibecoding 101
Pre-Workshop Setup

Get ready to
actually build something.

Before we meet, set up the tools below. This takes about 30–45 minutes. Come with everything ready — we won't have time to troubleshoot installs on the day.

1

GitHub — Your Code Home

Where your projects live. Free forever.

GitHub stores your code and lets you deploy with one click. Every project you build at this workshop will live here.

1
Create a free account

Go to github.com and sign up. Use a professional username — this is your public developer identity.

→ github.com/signup
2
Install Git on your machine

Download and install the Git CLI. This is what connects your computer to GitHub.

→ git-scm.com/downloads
3
Verify it works
# Run this in Terminal / Command Prompt
git --version
# Should print something like: git version 2.x.x
💡 Tip: Already have GitHub? Just make sure you remember your username and password. You'll need them when connecting to Vercel and Convex.
2

Vercel — Deploy in Seconds

Free hosting for your web apps. No config needed.

Vercel takes your code from GitHub and turns it into a live URL in under 30 seconds. This is how you share what you build with the world.

1
Sign up with GitHub

Go to vercel.com and click "Sign Up" — choose "Continue with GitHub". This links the two together automatically.

→ vercel.com/signup
2
Install the Vercel CLI (optional but useful)
npm install -g vercel
# Then login:
vercel login
💡 Tip: The free Hobby plan is all you need. Don't upgrade anything.
3

Convex — Your Backend & Database

Real-time database that AI can write. Free tier is generous.

Convex is a backend-as-a-service that works perfectly with AI coding tools. Instead of setting up a database manually, Convex handles it for you — and it works really well with Claude Code out of the box.

1
Create a free account

Sign up at convex.dev — you can use GitHub to sign up.

→ dashboard.convex.dev/signup
2
Install the Convex CLI
npm install -g convex
# Verify:
npx convex --version
3
Make sure Node.js is installed

Convex requires Node.js. If you don't have it:

→ nodejs.org (download LTS)
💡 Why Convex? Most beginners get stuck setting up databases. Convex removes that entirely — you describe your data, it handles the rest. It's the fastest path from idea to working app.
4

Your AI Coding Tool

Claude Code recommended — alternatives accepted.

This is the AI that will help you build. Pick one and have it set up before you arrive. The workshop will be demoed using Claude Code via Antigravity in the terminal — but any of these will work for the exercises.

Claude Code ★ Recommended
What we'll be demoing live. Best for complex apps, understands context deeply. Runs in your terminal. Needs Anthropic API access or Antigravity subscription.

→ Get Claude Code
Cursor Popular
VS Code with AI built in. Great for beginners. Has a free tier. Bring your own API key or use theirs.

→ cursor.sh
Windsurf Alternative
Similar to Cursor. Good free tier. Works well for web projects.

→ codeium.com/windsurf
Lovable / Bolt No-code
Browser-based, easiest to start. Good for prototyping. Less control than the others but zero setup required.

→ lovable.dev
💡 On models: Claude Opus gives the best results for complex app logic. If you have an existing Claude Pro subscription, you already have access. If budget is tight, Claude Sonnet or Cursor's free tier will work fine for the workshop exercises.
5

Come with an Idea

No idea? We'll figure it out together.

The workshop is hands-on. You'll be building something real, not following a pre-made tutorial. Come with one of these:

A
An existing project that's stuck or broken

Perfect. We'll use it as a live case study. Bring the repo link or the codebase on your machine.

B
A new idea you want to start building

Think: a simple web app, a tool that solves a problem you have, a side project you've been sitting on. Keep it focused — one core feature is enough to start.

C
Nothing yet — that's okay

We'll pair you with a starter project and you'll still walk away having built and deployed something real.

💡 Best ideas for beginners: A personal dashboard, a simple booking form, a landing page with a waitlist, a habit tracker, or a tool that automates something you do manually.

Your Setup Checklist

GitHub account + Git installed
Vercel account connected to GitHub
Convex account + CLI installed
AI coding tool installed & logged in
Idea or project ready to go

All set? You're ready. See you at the workshop. 🚀
